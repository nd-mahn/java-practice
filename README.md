# 🚀 Java Learning Workspace

## 📌 Giới thiệu

Đây là workspace dùng để **học Java từ Core → Spring** theo hướng:

* Học theo module
* Mỗi bài có **lý thuyết + thực hành**
* Tập trung vào **hiểu bản chất + áp dụng thực tế**

---

# 🧱 Cấu trúc Project

## 1. 📦 JavaCoreProject

```text
java-core-project/
 ├── notes/                  # Lý thuyết từng phần
 ├── src/main/java/...       # Code thực hành
 └── resources/              # File input/output
```

### 📚 Nội dung học

* JVM & Memory
* OOP (Abstract, Interface, Polymorphism)
* Collections & Generics
* Java 8 (Stream, Lambda)
* Exception Handling
* Concurrency (Thread, Lock)
* File I/O
* JDBC
* Network (Socket)

### 🎯 Mục tiêu

* Nắm vững nền tảng Java
* Hiểu cách code hoạt động
* Debug được lỗi phổ biến

---

## 2. 🌱 SpringMasteryProject

```text
spring-mastery-project/
 ├── notes/                  # Lý thuyết từng module Spring
 ├── src/main/java/...       # Code thực hành
 ├── resources/              # Config (yml, sql,...)
 └── Dockerfile / compose    # Deploy
```

### 📚 Nội dung học

* Spring Core (IoC, DI)
* Bean Lifecycle & Scope
* Spring JDBC
* Hibernate ORM
* Spring Data JPA
* Spring AOP
* Spring Web (REST API)
* Validation & Exception
* Async & Scheduling
* Security (JWT)
* Caching
* Actuator & Monitoring
* Docker
